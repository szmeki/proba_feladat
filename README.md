# Quantum

## Developing

First install node 16 with nvm. In a shell (Git Bash on Windows) enter:

```
# install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
# install node
nvm install v16
```

Then install postgresql database

Then in this folder: 

```
npm install
npm run dev
```
